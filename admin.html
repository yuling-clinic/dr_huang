🐻 後台編輯

在這裡編輯網站所有內容。設定好 GitHub 連線後，按「發布到網站」就會直接更新。編輯過程會自動暫存在這台裝置。

GitHub 連線（設定一次即可）

如何取得 Token？
  1. GitHub → 頭像 → Settings → 最下方 Developer settings
  2. Personal access tokens → Fine-grained tokens → Generate new token
  3. Repository access 選「Only select repositories」，勾選您網站的 repo
  4. Permissions → Repository permissions → Contents 設為 Read and write

網站基本資訊

看診資訊

每個看診地點（醫院／診所）建一筆，填敘述文字＋掛號連結，下面可以再加這個地點專屬的門診時段。

文章管理

編輯器像 Word 一樣操作。要連到自己的其他文章，按「📎 站內文章」直接挑選，不用管網址。

📊 網站數據

看有多少人瀏覽、哪幾篇最多人看。兩個都是免費的，且不使用 cookie、不追蹤個人身分。

設定（第一次使用請點開）
如何取得？
  1. 到 dash.cloudflare.com 註冊免費帳號（不用轉移網域）
  2. 左側選單 → Analytics & Logs → Web Analytics
  3. 點「Add a site」，輸入你的網站網址
  4. 它會給你一段程式碼，裡面有一串 "token": "xxxxx"
  5. 把引號中間那串 xxxxx 複製貼到上面欄位，然後重新發布網站即可
如何取得？
  1. 到 search.google.com/search-console
  2. 新增資源 → 選「網址前置字元」→ 貼上你的網站網址
  3. 驗證方式選「HTML 標記」，它會給你一段 <meta name="google-site-verification" content="xxxxx">
  4. 把 content 裡面那串 xxxxx 貼到上面欄位 → 重新發布網站 → 回 Google 按「驗證」
  5. 驗證通過後，左側「Sitemap」提交 sitemap.xml

發布到網站

會把所有內容寫回 GitHub，並為每篇文章產生獨立的網頁檔（讓 Google 搜尋得到），同時更新 sitemap。網站約 1–2 分鐘後更新。

選擇要連到的文章

請先在內文中選取要變成連結的文字，再點這裡挑選文章。
